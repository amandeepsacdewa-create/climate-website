This page explains the design and development of the pages created by me for the Climate Action website.

Home Page

The home page was designed using a clean and simple layout with a consistent green colour scheme to match the Climate Action theme. A navigation bar was included at the top to allow users to easily move between pages. Card elements were used to present key climate topics in a visually organised way.

Gallery Page

The gallery page displays a collection of climate-related images. JavaScript was used to create an interactive feature where users can click on an image to view a larger version in a popup. This improves user experience and engagement. CSS styling ensures the layout is responsive and visually consistent.

Content Page

The content page focuses on technology and climate solutions. It includes sections on renewable energy, smart cities, electric transport, and carbon capture. Internal navigation links allow users to quickly jump to different sections, and images are used to support the information presented.

Sitemap

The sitemap page provides a visual overview of the website structure, showing how all pages are connected. This helps users understand the layout of the site and improves navigation. It also demonstrates planning and organisation during the development process.

Accessibility

Accessibility features include the use of descriptive alt text for images, clear headings, and readable font sizes. The website layout is responsive, ensuring it works well on different screen sizes and devices. Colour contrast was also considered to improve readability.

Validation

All pages were validated using the W3C Markup Validation Service. Errors such as missing attributes and incorrect HTML structure were identified and fixed to ensure the website follows proper web standards.

Reflection

Developing these pages improved my understanding of HTML, CSS, and JavaScript. One challenge I faced was implementing the image popup in the gallery, as it initially displayed incorrectly. This was resolved by adjusting the CSS positioning and structure. Overall, this project helped me develop both technical and problem-solving skills.
